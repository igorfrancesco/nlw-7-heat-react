### ⚙️ Tecnologias

Esse projeto foi feito com as seguintes tecnologias.

- ReactTS
- TypeScript
- SCSS
- NodeJS
- Prisma
- Socket.io

### 📖 Projeto

Aplicação fullstack feita no NLW para o evento DoWhile 2021. A aplicação contém comunicação real time com o servidor node e integrações websockets.
